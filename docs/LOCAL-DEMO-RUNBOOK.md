# OverdueFree 本地演示手册

本文档用于在 Windows 本地电脑上给朋友或少数试用人员演示第一版闭环。

## 1. 演示前准备

1. 确认 Docker Desktop 已启动。
2. 双击根目录 `start-local-demo.bat`，或按顺序启动：
   - `start-mysql.bat`
   - `backend/start-backend.bat`
   - `start-frontends.bat`
3. 等待后端窗口提示健康检查通过。
4. 打开客户 H5：`http://localhost:5173/#/pages/home/index`
5. 打开后台 Web：`http://localhost:5174`

## 2. 默认后台账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 老板 | boss | boss123456 |

## 3. 最小演示路径

1. 在 H5 输入测试手机号，例如 `18600000001`。
2. 点击获取验证码，页面会显示本次模拟验证码。
3. 使用验证码登录，进入首页。
4. 点击 `AI债务咨询师`。
5. 按聊天流程填写称呼、地区、债务金额、债务类型和补充说明。
6. 提交后查看企业微信二维码弹窗。
7. 登录后台。
8. 进入 `线索管理`，确认能看到刚刚登录的手机号、提交的线索信息和二维码查看行为。
9. 按条件筛选线索。
10. 勾选导出字段，点击 `导出 Excel`。
11. 打开导出的 Excel，确认手机号、线索、行为和时间字段可读。

## 4. 可选演示点

1. 首页视频：后台上传 mp4 后，H5 首页可点击播放。
2. 素材管理：后台替换首页入口图、视频封面、企业微信二维码。
3. 规划评估：从首页进入规划表单，提交后后台会新增一条线索记录。
4. 真实利率计算器：只在前端本地计算，不保存明细。
5. 案例和资讯：后台新增或编辑后，H5 列表和详情可查看。
6. 管理员管理：老板账号可新增普通管理员，普通管理员不可管理管理员账号。

## 5. 常见卡点

### 5.1 H5 提示无法连接后端

确认后端窗口仍在运行，并访问 `http://localhost:8080/api/health` 检查健康状态。

### 5.2 后台登录 403 或登录失败

确认后台请求地址指向本地后端 `8080`，并重启后台开发服务以加载代理配置。

### 5.3 上传文件后 H5 没有变化

1. 确认后台保存成功。
2. 刷新 H5 页面。
3. 确认上传目录为 `C:/OverdueFree/uploads/`。

### 5.4 端口被占用

当前脚本不主动检查或释放端口。请查看对应启动窗口提示，再决定是否关闭占用进程或改端口。

## 6. 演示口径

1. 这是本地演示版，不接真实短信，不接真实大模型。
2. 页面上的债务方案、案例、资讯均为演示内容，正式上线前需要业务方替换。
3. 二维码承接当前为展示企业微信二维码，后续可扩展为真实企业微信跳转。
4. 所有时间均按北京时间展示。
